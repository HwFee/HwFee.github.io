常见问题

Q: 找不到 openclaw 命令

A: npm 全局 bin 目录可能不在 PATH 中。重启终端或手动添加：

Q: 网关无法启动

A: 检查端口占用，或运行 openclaw doctor --fix 自动修复。

Q: 模型调用失败（401）

A: API Key 未设置或错误。用 openclaw models auth list 查看，用 openclaw models auth set <provider> <key> 设置。
